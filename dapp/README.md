# DApp

![DzApp screenshot](../images/DApp.png)  

To run the server :

```bash
$ export FLASK_APP=server.py
$ python3 -m flask run --host=0.0.0.0
```


## Developpement mode

You can also run the server as this for developping. Code auto reload and logging included :)

```bash
$ python3 server.py
```
